           The  ASxxxx Cross Assembler and Linker package (V5.10 October
        2014) contains cross assemblers for  the  1802,  S2650,  MPS430,
        6100,  61860,  6500,  6800(6802/6808), 6801(hd6303), 6804, 6805,
        68HC(S)08, 6809, 68HC11, 68HC(S)12, 68HC16, 740,  78K/0S,  8048,
        8051,  8085(8080),  AVR,  DS8xCxxx,  ez80,  F2MC8L/FX,  F8/3870,
        GameBoy(Z80), H8/3xx, Cypress PSoC(M8C), PIC, Rabbit  2000/3000,
        SC/MP,   asst6,   asst7,  asst8,  z8,  and  z80(hd64180)  series
        microprocessors.  Complete source code is provided with the  as-
        sembler/linker submission.  





           The  ASxxxx  Cross  Assembler and Linker package is available
        from the server http://shop-pdp.net/.
